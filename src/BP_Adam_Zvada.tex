\abstractEN{The aim of this bachelor thesis is to explore the field of speech recognition and neural networks with a goal to implement end-to-end speech recognizer as voice-user interface for Robot NAO.

The proposed speech recognizer architecture is consisted from three main components: feature extraction of speech signal using mel-frequency cepstrum coefficients method, recognizer as recurrent neural networks with long-short-term-memory cells, and connection temporal classification algorithm for the final transcription.

The result of this work is end-to-end speech recognizer trained on VCTK corpus and implemented in programming language Python, using deep learning library TensorFlow.}
